Description

Provides roles to help managing Skupper sites

Author:

  • Fernando Giorgetti <fgiorgetti@gmail.com>

Supported ansible-core versions:

  • 2.14.0 or newer

Issue Tracker Homepage Repository (Sources)

Plugin Index

These are the plugins in the skupper.network collection:

Modules

  • links module – Update links based on provided links list

  • links_load module – Loads existing links to other sites

  • services module – Update services based on provided services list

  • services_load module – Loads existing services and targets

  • site_load module – Loads site information as ansible facts into the respective host
